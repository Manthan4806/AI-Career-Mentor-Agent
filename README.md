#  AI Career Mentor Agent

## Overview

AI Career Mentor Agent is an Agentic AI application built using Streamlit and Google's Gemini API. Unlike a traditional chatbot, it follows a multi-step workflow involving planning, tool selection, memory, and AI reasoning to provide personalized career guidance.

## Features

* Career Goal Detection
* Career Roadmap Generator
* Skill Gap Analysis
* Memory using Streamlit Session State
* Multi-step Agent Workflow
* Gemini AI Integration
* Conversation History
* Interactive Streamlit Interface

## Tech Stack

* Python
* Streamlit
* Google Gemini API
* python-dotenv
